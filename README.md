To run this Application, use the commands below:

>>> Git clone https://github.com/bonafide-ptnguyen/CertRoot.git

For Backend:
>>> cd backend
[install python]
[unix/linux] source .venv/bin/activate
[Windows] source .venv/Scripts/activate
pip install web3
pip install blake3
pip install pymongo
pip install dotenv
>>>  python certifier_integration.py
>>> Python app.py

For Frontend:
>>> cd frontend
>>> npm install
>>> npm run dev
